# telegram-purpose-bot

A Telegram bot designed for a specific purpose as requested by the user. Note: The prototype will focus on the bot's conceptual design and user interaction flow. Actual backend functionality for Telegram bots is not supported, but this will provide a solid foundation for further development.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/telegram-purpose-bot.git
cd telegram-purpose-bot
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
